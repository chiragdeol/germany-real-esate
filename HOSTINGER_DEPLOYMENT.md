# Hostinger Deployment Guide for European Deal Connect

This guide explains how to deploy the pre-built `hostinger-deploy.zip` file to Hostinger using their **Node.js application hosting**.

Since this project has server-side APIs (like the AI chat concierge), it must be deployed as a **Node.js Application** rather than a static site.

---

## Prerequisites
1. A Hostinger hosting plan that supports Node.js (e.g., Cloud hosting plans, Business plans, or a VPS).
2. The `hostinger-deploy.zip` file located in the root of your project folder.
3. Your **Lovable API Key** (required for the AI concierge chat feature).

---

## Step 1: Create the Node.js Application in Hostinger hPanel

1. Log in to your **Hostinger Account** and go to your website's control panel (**hPanel**).
2. On the left sidebar, navigate to **Advanced** > **Node.js** (or search "Node.js" in the top search bar).
3. Click the **Create Application** button.
4. Fill in the following settings:
   - **Application path:** Enter a folder name where your code will reside (for example: `deal-connect` or `public_html/deal-connect`).
   - **Node.js version:** Choose `20.x` or `18.x` (latest stable).
   - **Application source file:** Enter `server.js` (this is the entry point file inside our ZIP).
   - **Application URL:** Select the domain or subdomain where you want the app to be active.
5. Click **Create**.

---

## Step 2: Upload and Extract your Files

1. Go to **Files** > **File Manager** in hPanel.
2. Navigate to the application path folder you created in Step 1 (e.g., `deal-connect`).
3. If there are any default files (like a default `app.js` or `index.js` generated by Hostinger), **delete them**.
4. Click **Upload** (the arrow icon in the top right) and select **File**.
5. Upload the **`hostinger-deploy.zip`** file from your local machine.
6. Once uploaded, right-click `hostinger-deploy.zip` and select **Extract**.
   - Extract it directly into the current folder (so that the files are placed directly in your application path folder, not inside a subfolder).
7. Delete the uploaded `hostinger-deploy.zip` file to keep the server clean.

You should now see the following structure inside your application directory:
```text
├── public/
├── server/
├── nitro.json
├── package.json
└── server.js
```

---

## Step 3: Install Dependencies

1. Go back to the **Node.js** dashboard in hPanel.
2. Click **Edit** or click on your newly created application.
3. Locate the **Run npm install** button (or **npm install**).
4. Click it and wait for a few seconds. This installs the light `tslib` helper package needed by the server.

---

## Step 4: Configure Environment Variables

For the AI concierge chat to work, you must add your API Key.
1. In the **Node.js** application settings in hPanel, locate the **Environment Variables** section.
2. Add the following environment variables:
   - **Key:** `LOVABLE_API_KEY`  
     **Value:** *[Your actual Lovable API key]*
   - **Key:** `NODE_ENV`  
     **Value:** `production`
3. Save the changes.

---

## Step 5: Start the Application

1. In the **Node.js** panel, click **Start** (or **Restart** if it is already running).
2. Visit your domain or subdomain. Your web app is now fully functional, including the AI concierge!
